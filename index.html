ABOUT

EZRANGER is an app that helps you create better estimates using a formula that takes into account any interruptions you may encounter during the course of a project.

INSTRUCTIONS

  1. Enter your Project Name.
  2. Break your Project into Tasks.
  3. Name each Task.
  4. Enter an Aggressive But Possible Estimate. This is how fast you think you can get each task done without any delays, interruptions or issues.
  5. Enter a Highly Probably Estimate. This should be a number you're 100% comfortable with, taking into consideration any delays or complexities.
  6. Choose if your estimates are in Days or Hours
  7. Press Calculate

EZRANGER is a project by Myke Preuss based of an formula seen on Atomic Spin pointed out by Ben Feist. Special thanks to William Kral and Raman Gill.

EZRANGER is on github and uses Bootstrap.
